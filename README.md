# Plot in C

Plotter de funções matemáticas em **C99** com renderização 2D em **SDL2** e parsing de expressões em sintaxe inspirada em **TeX/LaTeX**.

## Requisitos (Arch Linux)
```bash
sudo pacman -Syu
sudo pacman -S --needed gcc make git sdl2
```

# Build

```bash
make
```

# Run (Hello SDL2)

```bash
make run
```
